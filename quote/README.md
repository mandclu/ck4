# Quote Widget

This widget wraps a blockquote element within a figure element and allows for an optional caption. The figure element will have the CSS class `quote`.

    <figure class="quote>
        <blockquote>
            ...
        </blockquote>
        <figcaption>
            ...
        </figcaption>
    </figure>
